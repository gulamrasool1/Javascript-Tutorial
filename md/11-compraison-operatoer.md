## COMPARISON OPERATOR
|OPERATOR|DESCRIPION|
|--------|----------|
|==|EQUAL TO|
|!=|NOT EQUAL|
|!==|NOT EQUAL VALUE OR NOT EQUAL TYPE|
|>|GREATER THEN|
|<|LESS THEN|
|>=|GREATER THEN OR EQUAL TO|
|<=|LESS THEN OR EQUAL TO|
---
>we can compare any thing with the help of this table and find a value in (true) or (false) so the pattern is
